A Look at Enumarable Methods

.group_by - A Useful Enumerable

April 9th 2015

What is an enumerable?

We've become acquainted with methods over the past few weeks. A method can be invoked on an array or hash to perform different operations. We can get more complex and add a number of different methods on top of one another. This can get messy and end up being a lot of code over time. What is a solution to this? Enumerables! Enumerables are essentially packaged methods into one. They perform a number of methods on an array or hash and give us a new output with the enumerable invoked. Making some sense? Let's check out one of my favorite enumerables in action to help clarify.

Let's check out .group_by

I'm sure you've had times in your life where you have a list of names that you've wanted grouped by alphabet letter. Wouldn't it be great if there was something you could simply do to the list so that it spit out a number of lists of the names that start with each letter of the alphabet? Let's take a look at an enumerable that can do just that - '.group_by'. I am going to take our cohort list from DBC and group the names by the first letter of each member's first name. First, lets look at the array that contains our whole cohort:

sf_group = ["Alex Cusack","Alex Kass","Andrew Donato","Andrew Dowd","Arielle Chen","Brian Kennedy","Carissa Blossom","Christopher William Kramlich","David Matthew Grotting","Ellis Marte","Emily Boerner","Eric Flores","Eric Saxman","Ian Andrew Harris","James Newman","Jeffrey Chang","Jennifer Jaochico","Josh Ullman","Karan Aditya","Le Tran","Michael Farr","Miche Weinberg","Patrick Betti","Ryan Posthumus","Tanay Arora","Travis R Siebenhaar","Vivek Poola"]

As you can see, this is simply an array that contains the individual members of our San Francisco cohort. Now lets invoke the .group_by method that will group the members of the group by the first letter of their first name. Here is what the code will look like. I'll break it down below.

sf_group.group_by {|name| name[0]}

So what exactly is this doing? The enumerable is saying 'lets take each individual name in the array and group it by the first letter - referenced by name[0]'. It then returns the group as a hash, where each key represents a letter of the alphabet that the names start with and a value which will be an array of the names that start with that letter. Check it out below to see what this enumerable will return!

{"A"=>["Alex Cusack", "Alex Kass", "Andrew Donato", "Andrew Dowd", "Arielle Chen"], "B"=>["Brian Kennedy"], "C"=>["Carissa Blossom", "Christopher William Kramlich"], "D"=>["David Matthew Grotting"], "E"=>["Ellis Marte", "Emily Boerner", "Eric Flores", "Eric Saxman"], "I"=>["Ian Andrew Harris"], "J"=>["James Newman", "Jeffrey Chang", "Jennifer Jaochico", "Josh Ullman"], "K"=>["Karan Aditya"], "L"=>["Le Tran"], "M"=>["Michael Farr", "Miche Weinberg"], "P"=>["Patrick Betti"], "R"=>["Ryan Posthumus"], "T"=>["Tanay Arora", "Travis R Siebenhaar"], "V"=>["Vivek Poola"]}

Wasn't that cool? We now have a nicely sorted list of groups of names that start with different alphabet letters.
